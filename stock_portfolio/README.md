
# Stock Portfolio Tracker

A simple Python project that calculates total investment value based on stock prices and quantity entered by the user.

## Features
- User can enter stock names and quantity
- Uses a dictionary to store stock prices
- Calculates total investment value
- Supports multiple stock entries
- Saves result in a text file

## Technologies Used
- Python

## Concepts Used
- Dictionary
- Loops
- Conditional Statements
- User Input
- File Handling

## Example Stocks
- AAPL
- TSLA
- GOOGLE
- AMZN

## How to Run
1. Open the project folder
2. Run the Python file

```bash
python stock_portfolio.py
```

## Output Example

```text
AAPL x 2 = 360
TSLA x 1 = 250

Total Investment Value = 610
```
