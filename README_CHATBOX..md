# Basic Chatbot

A simple rule-based chatbot built using Python.

## Features
- Responds to simple user inputs
- Interactive console chatbot
- Supports multiple predefined replies
- Exit option using "bye"

## Technologies Used
- Python

## Concepts Used
- if-elif statements
- while loop
- functions
- user input/output

## Supported Commands
- hello
- how are you
- what is your name
- bye

## How to Run

```bash
python basic_chatbot.py
```

## Example Output

```text
🤖 Simple Chatbot

You: hello
Bot: Hi!

You: how are you
Bot: I'm fine, thanks!

You: bye
Bot: Goodbye!
```
